## Instalación

### Getting Started

> To deploy your application on-premises, you must follow the steps below.
>
> > Clonar el repositorio

```sh
 git clone https://github.com/juanRCode/Project-FAQ-accordion.git
```

> > Instalar el paquete de modules (npm)

```sh
 npm install
```

> > Visualizar la app

```sh
// mode development
 npm run dev
// final mode
 npm run start
```
